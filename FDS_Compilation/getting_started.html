Getting Started using FDS/Smokeview Build Scripts

Initial Setup

  • Defining environment variables used by the scripts
    1. Open the Utilities\Scripts folder and double-click on the file COPY_fds_smv_env.bat. This copies fds-smv_env.bat to your home directory.
    2. Click here to update the fds_smv_env.bat configuration file. (Define your FDS/Smokeview repository location.)

  • Compiling software
    1. Install Intel Fortran.
    2. Install Intel C/C++ (only required if you wish to build Smokeview)

  • Communicating with other Computers
    • Install putty.
    • Set up ssh keys. (optional - to avoid typing passwords)
      1. run puttygen
      2. click on the SSH-2 RSA radio button
      3. click on the Generate button. Pick a pass-phrase.
      4. Save the private key on your local computer.
      5. Save the public key and copy it to the Linux cluster. Put the key at the end of the file authorized_key2 which is located in the .ssh directory in your home directory.

  • Creating Bundles
    1. Install Winzip version 12, Winzip command line tool and Winzip self extractor. Note, The Winzip command line tool is free but is a separate download from Winzip 12. The Winzip self extractor needs to be purchased separately.
    2. Put winzip tools in c:\bin\winzip

  • Uploading bundles to googlecode
    1. Download and install Python
    2. Create a file named gc.passwd containing your google password and put it in your windows home directory.

Testing the Setup

  • Click here to verify that software tools needed by the FDS-Smokeview web scripts are installed.
