Initial Setup

  • Edit configuration file Define environmental variables used by the scripts on this page (such as svn revision numbers, user names, local SVN repository location, etc.).
  • Intel Compilers
    1. Install Intel Fortran and C/C++ compilers - version 11
    2. Put Intel environment def scripts in c:\bin (copy iclvars.bat and ifortvars.bat from the Utilities/Scripts directory to c:\bin)
  • Putty
    1. Install putty. Doesn't matter where - just make sure the install directory is in the path
    2. set up putty to enable no password logins (use pageant)
  • put google password in the file c:\bin\gc.passwd - this is needed for uploads to google
  • Winzip
    1. Install Winzip version 12, Winzip command line tool and Winzip self extractor. Note, The Winzip command line tool is free but is a separate download from Winzip 12. The Winzip self extractor needs to be purchased separately.
    2. Put winzip tools in c:\bin\winzip
