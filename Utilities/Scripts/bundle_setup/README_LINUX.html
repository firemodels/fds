Linux Install Notes

Latest version of these notes

These are preliminary notes for an upcoming release of the Linux version of FDS and Smokeview.

Future releases will use an installer. In addtion to copying files to user specified locations the installer will add the following lines

  • source .cshrc_fds
  • source .bashrc_fds
to the .cshrc and .bashrc startup files respectively. This is done in order to update the PATH and LD_LIBRARY_PATH environment variables allowing one to run FDS and Smokeview from the command line.

To use the parallel version of FDS (fds_mpi_linux_64 or fds_mpi_linux_32), OpenMPI libaries must be installed. To install:

  • Download the 32 or 64 bit (or both) Openmpi libraries using the following links.
    • OpenMPI 1.4.3 32 bit Linux
    • OpenMPI 1.4.3 64 bit Linux
  • Uncompress and untar the downloaded library archive placing the library contents /shared/openmpi_32 or /shared/openmpi_64
    (as root cd to /shared and type tar xvf openmpi32_linux_1.4.3.tar.gz or tar xvf openmpi64_linux_1.4.3.tar.gz )
