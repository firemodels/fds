OSX Install Notes

Latest version of these notes

The parallel (MPI) versions of FDS running under OSX require openmpi libraries to link to at runtime. These libraries may be obtained from the FDS/Smokeview download site.

Installing the Openmpi libraries

  • Download the 32, 64 or both Openmpi libraries using the following links.
    • openmpi32_osx.tar.gz - 32 bit OSX Openmpi 1.3.3 libraries
    • openmpi64_osx.tar.gz - 64 bit OSX Openmpi 1.3.3 libraries
  • Uncompress and untar the downloaded library archive.
  • Define the environmental variable DYLD_LIBRARY_PATH to point the lib directory within the library.

Examples

  • If you use the bash command shell (default on the Mac) add the following lines to your .bash_profile file
    export DYLD_LIBRARY_PATH=~/openmpi-1.3.3_32/lib
    export PATH=~/openmpi-1.3.3_32/bin:$PATH
  • If you use the csh or tcsh as your command shell add the following linee to your .cshrc file
    setenv DYLD_LIBRARY_PATH ~/openmpi-1.3.3_32/lib
    setenv PATH ~/openmpi-1.3.3_32/bin $PATH If you installed the 64 bit Openmpi libraries, change 32 to 64 in the above.
