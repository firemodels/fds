These notes are for an experimental installer used for the upcoming version 6 of FDS and Smokeview.

The FDS/Smokeview installer creates two files, .cshrc_fds and .bashrc_fds. These files add entries to the PATH and LD_LIBRARY_PATH environment variables allowing one to run FDS and smokeview from the command line.

To activate the statements found in .cshrc_fds and .bashrc_fds add

  • source ~/.cshrc_fds to .cshrc (if you use the csh or tcsh shell)
  • source ~/.bashrc_fds to .bashrc (if you use the sh or bash shell)
