Verification Template

This web page gives an example of how Smokeview scripting may be used to dynamically update web pages in order to examine or verify FDS results as FDS and/or Smokeview is updated. To adapt this to your own case, simply edit scripts to run your own case and create a Smokeview script that generates the images you wish to examine.

Generate Images

The following two links run scripts that assume that the FDS repository is located at d:\fds-smv on your PC. Edit these two scripts (located in the scripts directory) if your repository is in a different location.
