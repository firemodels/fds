Bundle Steps - Smokeview

  1. Update repos on all three platforms
  2. Edit configuration file labeling the bundle name
  3. Update smokeview release notes:
    1. Add smokeview log entries to release notes (for commits made after smvlogdate defined in config file)
    2. cleanup log entries and commit ( view release notes )
  4. build libraries on all three platforms
  5. build smokeview (after library builds complete)
  6. build Utilities (after library builds complete)
