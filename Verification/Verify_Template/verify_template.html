Verification Using Scripting

This web page gives an example of how scripting may be used to help examine FDS results. The use of scripts guarantee that results from one run to the next are consistent (same view points, same time etc.)

Windows batch files (scripts) are used to run FDS and Smokeview.

A Smokeview script is used to generate the images. These images may then be easily regenerated as FDS, Smokeview and/or the input file changes in order to examine the effect of these changes.

To adapt this example for your own case, edit the makeimages.bat and runcase.bat files located in the scripts directory to point to the case being examined.
