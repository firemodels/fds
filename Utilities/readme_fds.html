Configure the Build Scripts

The aim of the scripts on this web page is to create files for updating FDS on a Windows, Linux and/or a Mac computer. These names are formed using the FDS version number and the SVN revision number of the FDS source.
  • Configure the FDS build scripts
  • Find the SVN revision number of the FDS source
  • Update the FDS source to the revision specified in the fds_smv_env.bat configuration file.
  • Update the FDS source to the latest revision.
  • Update all repositories (Windows, Linux, Mac) to the latest revision..
