Initial Setup

  • Setting up environment defintions
    1. Copy fds_smv_env.bat to your home directory - Do this by opening up the FDS_Compilation\Scripts folder and double-clicking on the COPY_fds_smv_env.bat batch file.
    2. Customize fds_smv_env.bat - Define keyword values for your own setup by editing values in fds_smv_env.bat.
      Note: this only needs to be done once. Example:
      set svn_root=d:\fds-smv
      set svn_drive=d:
      set linux_svn_root=FDS-SMV
      set linux_username=gforney
      set google_username=gforney
      set linux_hostname=acrux.cfr.nist.gov

      Note: other edits will be made when you start building FDS or Smokeview (specifying version strings, revision numbers etc).

  • Compilers
    1. Install Intel Fortran and (optionally) C/C++ compilers - version 11
    2. Copy the compiler environmental setup scripts into c:\bin. (Create c:\bin if it doesn't exist already.)
      1. Intel Fortran: copy ifortvars.bat from the Intel compiler bin directories to c:\bin
      2. Intel C/C++: copy iclvars.bat from the Intel compiler bin directories to c:\bin . Note this step is only necessary if you build Smokeview using these scripts. In that case, you MUST also install the Intel C/C++ compilers.
      3. Visual Studio C/C++: Open up the FDS_Compilation/Scripts directory and double click on the COPY_clvars.bat batch file. This will copy clvars.bat from the repository to c:\bin.

        Edit clvars.bat to make sure the path specified in this batch file matches your distribution of Visual Studio.

  • Putty
    1. Install putty. Doesn't matter where - just make sure the install directory is in the path .
      Test the installation by opening up a command shell and typing plink at the command line.
    2. Set up ssh keys to let you login into the Linux cluster without a password. Basically you are creating a public/private pair of ssh keys. You copy the private key to somewhere on your PC (doesn't matter where) and copy the public key into a file on the Linux cluster. More precisely:
      1. run puttygen
      2. click on the SSH-2 RSA radio button
      3. click on the Generate button. Pick a pass-phrase that you can easily remember.
      4. Save the private key into the c:\bin\keys directory (create the directory it if it doesn't already exist)
      5. Save the public key and copy it to the Linux cluster. Put the key at the end of the file authorized_key2 which is located in the .ssh directory (in your home directory)
    3. On your PC, copy the file Pageant.exe to C:\Documents and Settings\gforney\Start Menu\Programs\Startup (change gforney to your user name).
    4. Reboot your computer.
    5. Open Pageant (an icon looking like looking like a terminal wearing a black hat should appear in the right side of the task bar). This program will direct you to open your private key and ask you for the pass-phrase used to create it.

  • Uploading files to googlecode with Python

    Download and install Python

  • Google password file

    Create a file named gc.passwd containing your google password and put it in your windows home directory.

  • Winzip
    1. Install Winzip version 12, Winzip command line tool and Winzip self extractor. Note, The Winzip command line tool is free but is a separate download from Winzip 12. The Winzip self extractor needs to be purchased separately.
    2. Put winzip tools in c:\bin\winzip
