GettingStarted
[[SVN How-To]]
Introduction

Google Code uses Subversion (SVN) for version control. Subversion is widely considered as an improved versioning method over CVS. This Wiki page provides instructions on using SmartSVN to obtain files from the FDS-SMV SVN Repository.
Obtaining SmartSVN

SmartSVN is available as freeware. A for-purchase version of SmartSVN is available that has additional capabilities, but for most users the free version will suffice. Note that other programs are available (list of some found here).
Adding the Repository to SVN

    * After installing SmartSVN, launch the program.
    * The program will open with a Welcome to SmartSVN window. Select Check out project from repository
    * A Check Out Project window will open. Select Manage
    * A Repositoty Pofiles window will open. Select Add
    * An Add Repositroy Profile window will open.
          o Select HTTP for the protocol
          o In the Server Name box type fds-smv.googlecode.com
          o In the Repository Path box type /svn/trunk
          o Click Next
          o Select Anonymous for the SVN Login
          o Click Next
          o If you desire to see something other than the URL in Check Out Project window select Use This Profile Name and enter a name
          o To check out the code follow the instructions in the next section 

Obtaining Files
The First Time

Once a project has been added to the repository it can then be checked out.

    * After launching SmartSVN, select Check out project from repository
    * A Check Out Project window will open. Using the drop down box select the FDS-SMV repository.
    * Click Next
    * SmartSVN will scan the repository in display a directory tree. The folders in the tree can be opened and closed with the plus/minus buttons. If you wish to maintain an archive of all FDS-SMV related files select the top level folder. If you wish to maintain seperate archives for portions of the FDS-SMV project enter the directory structure's trunk folder and select that portion you wish to obtain.
    * Click Next
    * Type in or browse a Local directory where the files will be stored.
    * Click Next 

After the First Time

    * After launching SmartSVN, select Open Existing project(s):
    * Select the Project Name you wish update
    * Click OK
    * You can now select either a file or a subdirectory and click the green arrow to update your local files from the repository
