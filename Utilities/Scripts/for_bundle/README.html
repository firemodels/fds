Installing Open MPI

  • To install Open MPI using the compressed tar file found in the FDS/Smokeview bundle
    1. copy openmpi_1.8.4.tar.gz to the desired directory location
    2. type: tar xvf openmpi_1.8.4.tar.gz
    3. edit the following line in the .bashrc file (or .bash_profile on the Mac) 
          source ./bashrc_fds path_to_openmpi_location
