Installing Open MPI

  • The OpenMPI compressed archive, openmpi_VERSION_PLATFORM_64.tar.gz, is located in the same directory containing fds where PLATFORM is either linux or osx and VERSION is the openmpi version string. The directory containing fds may be found by typing 
    which fds
    on a command line. To install OpenMPI:
    1. copy openmpi_VERSION_PLATFORM_64.tar.gz to the desired OpenMPI location
    2. cd to that directory
    3. type: tar xvf openmpi_VERSION_PLATFORM_64.tar.gz
    4. edit the following line in the .bashrc file (or .bash_profile on the Mac) so that the argument contains the path pointing to the OpenMPI location. 
          source ~/.bashrc_fds path_to_openmpi_location

  • To install in the fds bin directory, type: 
    cd path_to_fds_bin_directory
tar xvf openmpi_VERSION_PLATFORM_64.tar.gz
    and add path_to_bin_directory to the source ~/.bashrc_fds line in .bashrc (or .bash_profile on the Mac)

Using a Different Intel Fortran Version

  • fds in this installation was built using Intel Fortran version 17. If you wish to build and run fds using a different Intel Fortran version pass a second argument to .bashrc_fds giving the location of the Intel shared libraries for that version. In general, the following should be in your .bashrc (Linux) or .bash_profile (Mac) startup file

    source ~/.bashrc_fds path_to_openmpi_library path_to_Intel_shared libraries

    The following line

    source ~/.bashrc_fdss /shared/openmpi_64 /opt/intel17/compilers_and_libraries/linux/lib/intel64

    would setup your system to build and run fds using an OpenMPI library located at /shared/openmpi_64 and version 17 of the Intel Fortran compiler.
