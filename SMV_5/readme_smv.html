Verify Smokeview

  • Run test cases Run FDS cases used to test smokeview (allow several hours to complete). This step does not need to be performed every time.
  • Create figures Run Smokeview scripts to generate figures for the User's Guide
  • Generate User's Guide Use pdflatex and bibtex to generate the User's guide
  • View User's Guide Examine the User's guide to verify that Smokeview is working properly.

Package and Upload a Release Smokeview

  • Preliminaries - Rebuild Smokeview and the User's guide.
    • Increment the version number (SMVVERSION found in options.h . Add a dummy comment in main.c to force a repository update.
    • IMPORTANT - Commit changes so that the proper revision number appears when Smokeview runs.
    • Rebuild a Windows release version of Smokeview.
    • Regenerate test cases and Smokeview user's guide using instructions above under Verify Smokeview. Rebuilding the User's guide ensures that the proper Smokeview version information appears in the guide.
  • Update release notes Add entry to release notes for Smokeview update about to be released.
  • Copy Users guide Copy the Smokeview User's guide to the All_PDF_Files directory for distribution.
  • Commit updated release notes and User's guide to the repository so that that the packagers have access to them on all platforms.
  • Update configuration file Update the smv_revision and smv_version environment variables in the c:\bin\fds_smv_env.bat file.
  • Compile and Package Smokeview for 32 bit windows
  • Compile Smokeview for 32 bit OSX
  • Compile Smokeview for 32 bit Linux
  • Package Smokeview for Linux and OSX
  • Upload Upload Smokeview installation/archive files to the download site.
  • Check download page

Package and Upload a Test Smokeview

Test smokeview's are generated to test a particular fix usually as a result of a bug reported in the issue tracker. The test suite is typically not run.
  • Package a test Smokeview Create a Smokeview installation file for Windows.
  • Upload Upload the Smokeview installation file to the download site.
