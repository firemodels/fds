Verify Smokeview

  • Run test cases Run FDS cases used to test smokeview (allow several hours to complete)
  • Create test figures Run Smokeview scripts that generate figures for the User's Guide
  • Generate User's Guide Use pdflatex and bibtex to generate a PDF version of the Smokeview user's guide
  • View User's Guide Examine the generated figures in the User's guide to verify that Smokeview is working properly.
