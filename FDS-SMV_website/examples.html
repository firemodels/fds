Examples

The Test Cases page provide links to FDS Input file in the SVN repository and an image or video of the results visualized in Smokeview for each case. More information about these files may be found in the FDS User Guide.

Examples of FDS and Smokeview features are shown on the Visualization Features page. FDS results are visualized with Smokeview by displaying time dependant animations of particle flow, 2D slice contours, 2D vector slices, isosurfaces and surface boundary contours. Data at a particular time may also be visualized using 2D/3D contours and vector plots.

In the Case Studies page, images and videos are used to demonstrate interesting comparisons between real world tests and simulations. Where possible, the FDS input file and/or test data or reports will be referenced.
