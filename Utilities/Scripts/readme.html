Configuring the Build Scripts

The aim of the scripts on this web page is to create installation files used to update FDS on a Windows, Linux and/or a Mac computer. These names are formed using the FDS version number and the SVN revision number of the FDS source.
  • Find the SVN revision number of the FDS source
  • Specify the FDS source revision number in the fds_smv_env.bat configuration file - used by all scripts run from this web page.
  • Update the FDS source to the revision specified in the fds_smv_env.bat configuration file.
  • Update the FDS source to the latest SVN revision number .
