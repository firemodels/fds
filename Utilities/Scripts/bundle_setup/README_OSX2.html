These are preliminary installation notes for the upcoming OSX version 6 release of FDS and Smokeview.

The new release uses an installer. In addtion to copying release files to the user specified location, the installer adds the following lines

  • source .cshrc_fds
  • source .bashrc_fds
to the .cshrc and .bashrc startup files respectively. This is done in order to update the PATH and DYLD_LIBRARY_PATH environment variables allowing one to run FDS and Smokeview from the command line.

To use the parallel version of FDS (fds_mpi_OSX_64 or fds_mpi_OSX_32), OpenMPI libaries must be installed. To install:

  • Download the 32 or 64 bit (or both) Openmpi libraries using the following links.
    • OpenMPI 1.4.3 32 bit OSX
    • OpenMPI 1.4.3 64 bit OSX
  • Uncompress and untar the downloaded library archive placing the library contents /shared/openmpi_32 or /shared/openmpi_64
    (as root cd to /shared and type tar xvf openmpi32_osx_1.4.3.tar.gz or tar xvf openmpi64_osx_1.4.3.tar.gz )
