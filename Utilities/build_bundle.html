Bundle - FDS - Smokeview - Guides

Bundle Steps

  1. Update all repositories
  2. Edit configuration file labeling the bundle name (version string for both FDS and Smokeview)
  3. Run firebot and smokebot (using firebot and smokebot accounts on blaze)
  4. Build FDS
  5. Build Smokeview and utilities
  6. Copy FDS and Smokview manuals to FDS_Guides directories on all three platforms
  7. Update FDS release notes, Convert FDS wiki to html, then commit
  8. Update and commit Smokeview release
  9. Copy FDS Smokeview guides from bot directories to to ~/FDS_Guides on all three platforms
