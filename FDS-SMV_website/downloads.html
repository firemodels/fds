Download Options

The links below are to listings of the latest Major or Minor release of the FDS-SMV Operating System specific packages. In the OS specific listing on the Downloads page, there may also be maintenance release files for these packages available. Maintenance release files are intended to patch/update the most recent Major or Minor release packages. You will need to download the Maintenance release files and use them to replace the files that were included in the OS specific Major/Minor release bundle.

For more information on Versions Numbers, Release Types and Revision Management see the Developer page.

FDS-SMV Operating System Specific Links

The links below will open the Downloads page on the FDS-SMV Google Code Project site, filtered to show only links to files related to the operating system selected.

Windows

The Windows file is a typical installer package that will install FDS, Smokeview and all associated files in the Program Files directory under the NIST subdirectory.

Apple OS X

The OS X file is a zip file, that contains a NIST folder, and a Read Me file that explains what to do with the folder.

Linux/Unix

The Linux/Unix file is a compressed tar file containing a NIST folder with all the binaries and documentation.

Source Files (Archive) or Individual Source Files (Web View)

Source files can be accessed as either a .zip archive file or individual files direct from the SVN repository web view. It is strongly recommended that you use the FDS-SMV Project SVN repository and a compatible SVN client to check out the latest source to your local machine and update it when necessary. This process will always reflect the most recent changes and will also allow you to retrieve a specific revision or taged version of FDS and Smokeview. See the Developer page for more information on accessing the repository.
